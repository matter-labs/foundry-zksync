Struct cast::revm::interpreter::OpCode

#[repr(transparent)]
pub struct OpCode(u8);
Expand description

An EVM opcode.

This is always a valid opcode, as declared in the opcode module or the OPCODE_INFO_JUMPTABLE constant.
